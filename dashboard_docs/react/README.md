# React / FastAPI Dashboard

This is the decoupled web architecture version of the dashboard. It relies on a Python FastAPI backend to serve and process data on the fly, while the modern React UI (powered by Vite) renders a premium user interface.

## Overview
- **Frontend**: React, Vite
- **Backend**: Python, FastAPI
- **Data Source**: Live connection to the local MLflow and Optuna SQLite databases.
