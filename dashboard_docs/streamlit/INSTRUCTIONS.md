# Instructions: Running the Streamlit Dashboard

Follow these simple steps to launch the Streamlit frontend.

### Step 1: Activate the Environment
Open a terminal in the root folder (`ml-cost-tracker`) and make sure your Python virtual environment is active.
```bash
# On Windows
.\.venv\Scripts\activate
```

### Step 2: Run Streamlit
Start the application by running the following command from the root directory:
```bash
streamlit run src/dashboard/app.py
```

### Step 3: View the Application
Once it starts, the application will automatically pop up in your default web browser, or you can manually navigate to:
👉 **[http://localhost:8501](http://localhost:8501)**
