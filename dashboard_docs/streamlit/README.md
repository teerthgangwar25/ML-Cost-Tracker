# Streamlit Dashboard

This is the primary Python-native dashboard built for the ML Cost Tracker. It allows you to visualize experiment convergence, explore hyperparameter importance (using SHAP), and restrict budgets to review optimal selections easily.

## Overview
- **Technology**: Streamlit, Plotly
- **Data Source**: Local SQLite tracking databases for MLflow and Optuna, alongside Parquet files generated by the DVC pipeline.

Make sure you have processed your data using `dvc repro` in the root folder before checking it out!
